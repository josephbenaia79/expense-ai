# Expense AI

AI-powered personal finance and expense tracking application built with Flutter and Firebase.

## Features

- User Authentication (Email & Google Sign-In)
- Expense Tracking
- Expense Categories
- AI Financial Assistant
- Firebase Firestore Integration
- User Profiles
- Modern Flutter UI

## Tech Stack

- Flutter
- Firebase Authentication
- Cloud Firestore
- Gemini AI API
- Dart

## Screens

- Login / Register
- Dashboard
- Add Expense
- AI Chat
- Profile

## Future Improvements

- Smart budgeting recommendations
- Expense analytics
- Financial forecasting
- Receipt scanning
- Investment insights
- Voice assistant
